About URL Shortner

A very simple public url shortner.
The backend API for creating new links is at /urls.
Here is a sample curl command to create a new link

curl -q -LSs -H "Content-Type: application/json" -X POST --data '{"url":"My_URL","slug":"My_URL_Name"}' https://gitsh.vercel.app/url;echo ''

Here is a sample function to create a new link

shrtn_link() { curl -q -LSs -H "Content-Type: application/json" -X POST --data '{"url":"https://$1","slug":"$2"}' https://gitsh.vercel.app/url 2>/dev/null && echo ''||return 2; }
